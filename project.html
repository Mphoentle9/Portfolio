Smart Lecturer Scheduling, Availability and Navigation System

A smart online and navigation system for the Mathematical and Computing Sciences department at Walter Sisulu University

Project Overview

The Smart Lecturer Scheduling, Availability and Navigation System is an innovative solution designed to address the common challenges students face when trying to locate lecturers on campus. By integrating IoT sensors, machine learning algorithms, and a user-friendly mobile application, the system provides real-time information about lecturer availability and optimal navigation routes and allows students to schedule appointments with their lecturers.

This system reduces time wasted searching for lecturers, improves communication between students and faculty, and enhances overall campus efficiency. The project was developed as part of our final year Computer Science capstone project.

Project Challenges & Solutions

Challenge 1: Accurate Indoor Positioning

Traditional GPS doesn't work effectively indoors, making navigation challenging.

Solution: Implemented a hybrid approach using Bluetooth beacons for proximity detection and Wi-Fi fingerprinting for more precise location tracking, achieving 92% accuracy in our tests.

Challenge 2: Real-time Data Synchronization

Maintaining consistent availability status across multiple devices and platforms.

Solution: Developed a WebSocket-based real-time communication system with a Redis cache layer to ensure instant updates across all connected clients.

Challenge 3: Privacy Concerns

Balancing tracking capabilities with lecturer privacy expectations.

Solution: Implemented an opt-in system with granular privacy controls, allowing lecturers to set availability preferences and location sharing limits.
